# 디자인 패턴 구현 
> Vue + TypeScript로 구현한 디자인 패턴 실습 과제 (영화관 예매 시스템)

---

## 프로젝트 개요
이 프로젝트는 Vue 3 + TypeScript를 사용하여  
영화 예매 과정 시나리오를 바탕으로 
**티켓 생성(Builder)**, **예매 절차(Facade)**, **결제 전략(Strategy)** 을  
각각 콘솔 기반으로 단순하게 구현했습니다.

---

## 디자인 패턴 요약

### Builder 패턴
- 복잡한 객체를 단계적으로 생성하는 패턴  
- 영화관, 영화, 시간, 좌석, 인원, 스낵 정보를 단계적으로 설정해 티켓 생성  

### Facade 패턴
- 여러 과정을 하나의 통합된 인터페이스로 단순화하는 패턴  
- 좌석 선택, 요금 계산, 예매 확정을 한 번에 수행  

### Strategy 패턴
- 알고리즘(행동)을 교체 가능하게 하는 패턴  
- 결제 방식을 카드, 간편결제, 무통장입금 등으로 변경 가능  

---

## 패턴별 설명
### 1. Builder Pattern — 티켓 생성

복잡한 티켓 정보를 단계별로 설정해 MovieTicket 객체를 구성
```
[Builder Pattern] 실행 시작
🎟️ [Builder] 티켓 생성 완료
MovieTicket { theater: 'CGV 전주고사', movie: '체인소맨: 레제편', ... }
```
### 2. Facade Pattern — 예매 절차 단순화

좌석 선택 → 요금 계산 → 예매 확정의 복잡한 과정을 하나의 인터페이스로 단순화
```
[Facade Pattern] 실행 시작
[Facade] 예매 절차 시작
💺 좌석 G8, G9 선택 완료
🍿 스낵(나쵸 콤보) 추가: +11,000원
👥 인원: 일반 2, 청소년 0, 우대 0, 장애인 0
💵 총 결제 금액: 41,000원
✅ 예매 확정: [CGV 전주고사] 체인소맨: 레제편 2025-11-08 19:30 / 좌석 G8, G9 / 41,000원
[Facade] 예매 완료!
```

### 3. Strategy Pattern — 결제 전략

결제 방식을 실행 중에도 교체할 수 있도록 구현.  
카드, 간편결제, 무통장입금 방식 중 하나를 선택해 결제 수행.
```
[Strategy Pattern] 결제 실행 예시
🎬 영화: 체인소맨: 레제편 @ CGV 강남
🕓 상영 시간: 2025-11-08 19:30
💺 좌석: G8, G9
🍿 스낵: 나쵸 콤보
💰 결제 금액: 41,000원
📱 간편결제로 41,000원 결제 완료
```

---

## 정리
| 패턴 | 목적 | 적용 내용 |
|------|------|-----------|
| **Builder** | 객체 생성 과정 단순화 | 티켓 정보 구성 |
| **Facade** | 복잡한 절차 통합 | 예매 과정 단일화 |
| **Strategy** | 행동(알고리즘) 교체 | 결제 방식 변경 |
